# Galactic Rust Defender

![Galactic Rust Defender](./download.jpeg)

Welcome to **Galactic Rust Defender**, a terminal-based, Galaga-inspired game written in Rust! Take control of your ship, fend off waves of enemies, and test your reflexes in this retro-style space shooter. 🚀✨

---

## Features

- **Classic Gameplay**: Fast-paced, nostalgic gameplay inspired by arcade classics.
- **Terminal Friendly**: Designed to run directly in your terminal, no GUI required.
- **Rust Powered**: Built with Rust for high performance and memory safety.
- **Customizable Controls**: Tailor your key mappings to suit your style.

---

## Installation
```bash
git clone https://github.com/GandalfTheGrey-DEV/Galaga.git
```
```bash
cd galaga
```
```bash
cargo run
```
