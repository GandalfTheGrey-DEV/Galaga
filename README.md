## 👾 About the Game

Galaga Rust Game is a fast-paced space shooter inspired by the iconic *Galaga*. Using Rust's performance and safety, this project brings retro arcade gameplay into the modern age.  
Dodge incoming alien waves, blast enemies, and aim for the high score!
