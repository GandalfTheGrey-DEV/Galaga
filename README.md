---

## 👾 About the Game

Galaga Rust Game is a fast-paced space shooter inspired by the iconic *Galaga*. Using Rust's performance and safety, this project brings retro arcade gameplay into the modern age.  
Dodge incoming alien waves, blast enemies, and aim for the high score!

### Features:
- 🛸 Smooth, real-time gameplay.
- 🌌 Retro-style graphics.
- 🔊 Original sound effects and music.
- 🎮 Keyboard and controller support.
- 🚀 Challenging alien wave patterns.

---

## 🛠️ Getting Started

### Prerequisites:
- Install [Rust](https://www.rust-lang.org/tools/install).
- Ensure you have `cargo` (Rust's package manager).
